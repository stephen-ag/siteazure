Interpolate Table v1.0

This app interpolates values for a given list from tabular input table .

Criteria used - Index interpolation based on look up table.

Home About
This web app will Interpolate the data based on input look up table. The criteria used is Index interpolation, where the data between two index is interpolated for all the columns.

Input Requirements: The input excel sheet should contain lookup table with sheet named "Input" and to look value in sheet named 'Interpolate_data'. Index column should be named as 'Time'

The Input sheet should contain row 2 starting with "Time,sec" as shown in picture and sheet showing "Interpolate_data" where data is placed for which interpolation required. Make sure the row number 2 should contain the all column heading for which interpolation is required





A new excel sheet will be created with the lookup table and the interpolated values and saved as interpolated file.

Select the input excel file using the Choose file button in the home page and submit the execute button, the output file will be saved
